Function:
Initial state:
Последние файлы, введенные в поиск, должны быть загружены, но поле search должно быть пустое.
----------
Plain-text search:
Запрос возвратил несколько свойств, приложение проверят одно или несколько свойств возвращены и перейти к рез поиска. Иначе еррор.
Если поиск неодзначен, то должен показываться список с предложенными местами
Когда поиск выполнен должен этот поиск в последние выведенные вывести на первое место.
----------
Listed locations state:
Список предлогаемых мест, когда будет нажато предлогаемое место, то будет обновлено текст с местоположением и поиск будет выполнен немедленно
Поиск должен происходить по ключу, а не по human-readable text.
----------
Error state and timeouts:
timout 5 sec
